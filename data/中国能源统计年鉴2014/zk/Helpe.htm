General Usage

 

One CD-ROM is attached with the China Energy Statistical Yearbook 2014, it is inseparable part of the yearbook. The CD-ROM is not permitted to be sold or bought separately.

No part of the CD-ROM may be stored on internet or intranet.

 

System Requirements：

Windows 98, Windows Me, Windows 2000, Windows XP, Windows 2003

Microsoft Internet Explorer 6.0 and above version

 

How to Use：

In order to use easily China Energy Statistical Yearbook 2014 for user, we provide two modes to browse; they are HTML form and EXCEL form. HTML mode is acquiescent. Click the linkage on every page; it can be interchanged between HTML and EXCEL.

1．              Insert China Energy Statistical Yearbook 2014 disc into your CD-ROM drive, the disc will be automatically started. If not, please click start.exe to open the CD-ROM.

2．              Entered the homepage, click Browse to view the yearbook content, click the table’s title to browse the relative statistical table, click Help  to open the help file, click Exit to quit the CD-ROM.

3．              Click the EXCEL/HTML at top left corner to interchange the files between HTML form and EXCEL form. Click Link Excel, Link Chinese and Full Window at the top of HTML mode to change to excel files, Chinese files or full window mode.

4．              Table's print and quotation. Forms can be printed and quoted under these two ways. Use the mouse to select printing area, right-click and select to print the current page, forms can be printed; Use the mouse to select printing area, click right key of the mouse, select copy, open the quotable applied program (such as Excel), select paste, selected table can fit Excel unit, but the form quoted from HTML has preset format, select excel edit-> delete-> delete form, can delete preset format.

5．              After open the Full Window browse, click back, can return to the browse interface.
